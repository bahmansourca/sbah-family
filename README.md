# SBah Family

A mobile application for managing family funds and events across different countries.

## Features

- User authentication and profile management
- Monthly contributions tracking
- Event management (weddings, funerals, etc.)
- Real-time transaction monitoring
- Multi-currency support
- Monthly reports generation
- Push notifications

## Tech Stack

- React Native with TypeScript
- Redux Toolkit for state management
- React Navigation for routing
- Axios for API calls
- React Native Elements for UI components

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- React Native development environment setup
- iOS Simulator (for Mac) or Android Emulator

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/sbah-family.git
cd sbah-family
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Run on iOS:
```bash
npm run ios
```

5. Run on Android:
```bash
npm run android
```
